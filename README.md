# outromanifesto.space - site

Site de divulgacao do livro *O Outro Manifesto Cypherpunk* (Lucas Lopes Galvao).
Astro + Tailwind, trilingue (PT/EN/DE). Deploy por tag `v*` -> GHCR -> VPS atras de Cloudflare Tunnel.

## Dev
    npm install
    npm run dev      # http://localhost:4321

## Build
    npm run build
    npm run preview  # node ./dist/server/entry.mjs
